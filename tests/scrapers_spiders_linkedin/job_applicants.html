      Principal Java Developer - Java Platform

      OraclePrague, Prague, Czech Republic

      2 weeks ago57 applicants

      See who Oracle has hired for this role

      The Java Platform Group is looking for an experienced, passionate, and highly motivated Java developer. Our team is responsible for delivering the Java platform that is used by millions of developers. We are looking for a seasoned engineer with a thorough understanding of both Java Platform and designing cloud native web services.

      We are looking for Java developer with strong technical background, who will design, implement and test our next generation cloud native services for remote management, monitoring and configuring JDKs. You will develop our stream processing backend, which will analyse telemetry of running Java applications. You will create software which will improve security and performance of JVMs.


      Skills we require:

      ·       10+ experience with architecting, implementing and testing large scale Java applications

      ·       5+ experience with architecting, implementing and testing Web Services in Java

      ·       Spring Boot/Helidon Web Services Framework

      ·       Experience with Jenkins CI/CD, Git, maven/gradle

      ·       API Documentation (Apiary / Swagger or similar tools)

      ·       Experience working on one or more major cloud vendor (OCI, Azure, AWS, GCP)


      Skills we love to have:

      ·       Cryptography, TLS knowledge, X.509 Certificates

      ·       Stream processing Kafka, Kinesis or similar stream processing technologies is big plus

      ·       Experience with performance tuning of Java (JFR, VisualVM, JProfiler, Parfait or similar technologies)

      ·       Experience with code analysis tools (FindBugs, SonarCube)

      ·       Networking using tools Terraform/Ansible

      ·       Knowledge of technologies like Docker, Kubernetes, Jaeger or Istio is advantage

      ·       Experience with SQL Databases (Oracle, MySQL, PostgreSQL)
